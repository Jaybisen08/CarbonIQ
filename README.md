# 🚀 Overview

CarbonIQ is a modern sustainability platform that enables users to understand and reduce their carbon footprint through data-driven analytics, AI-powered recommendations, and personalized sustainability goals.

Unlike traditional demo-based dashboards, CarbonIQ generates insights using **real user-entered data**, making every report and recommendation unique to the individual.

---

Calculate emissions based on:

* 🚗 Transportation

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/your-username/carboniq.git
```

## Navigate to the project

```bash
cd carboniq
```

## Install dependencies

```bash
npm install
```

## Configure environment variables

Create a `.env.local` file:

```env
NEXTAUTH_SECRET=

NEXTAUTH_URL=

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

SUPABASE_URL=

SUPABASE_ANON_KEY=
```

## Run the development server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```
