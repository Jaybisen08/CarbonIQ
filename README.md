# 🌱 CarbonIQ

### Measure Today. Improve Tomorrow.

CarbonIQ is a modern carbon footprint intelligence platform that helps individuals understand, track, and reduce their environmental impact through data-driven insights, sustainability goals, and personalized recommendations.

Built with a focus on usability, analytics, and environmental awareness, CarbonIQ transforms everyday lifestyle choices into actionable sustainability metrics.

---

## 🚀 Features

* Transportation
* Electricity Usage
* Food Habits
* Lifestyle Activities

* Sustainability Score

## ⚡ Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/carboniq.git
```

### Navigate to Project

```bash
cd carboniq
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
NEXTAUTH_SECRET=
NEXTAUTH_URL=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

SUPABASE_URL=
SUPABASE_ANON_KEY=
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
